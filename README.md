# Learning Fundamentals: 10-Week Backend & CS Deep Dive

A structured 10-week project-based learning plan covering CS fundamentals, backend depth, and cloud/DevOps. Each week builds a real system to understand concepts, not just memorize them.

## Projects

### Phase 1: CS Fundamentals (Weeks 1-3)
- **Week 1:** Search Engine Benchmark Tool – Big O analysis and complexity
- **Week 2:** Custom Data Structures – Hash tables, linked lists, binary search trees
- **Week 3:** DP Problem Solver – 8 classic dynamic programming problems

### Phase 2: Backend Depth (Weeks 4-6)
- **Week 4:** E-Commerce Database – SQL schema design and complex queries
- **Week 5:** Cache Layer – Redis integration and performance optimization
- **Week 6:** Production API – Authentication, rate limiting, validation

### Phase 3: Cloud/DevOps (Weeks 7-10)
- **Week 7:** Docker Containerization – Multi-stage builds and docker-compose
- **Week 8:** AWS Deployment – EC2 setup and infrastructure
- **Week 9:** CI/CD Pipeline – GitHub Actions automation
- **Week 10:** Managed Databases – RDS and disaster recovery
